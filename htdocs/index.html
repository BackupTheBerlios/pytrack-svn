Screenshots

24 October 2004

There are now two screenshots of the current status on the project website. One shows the map of a stored track, the other a distance/altitude graph. I also added a lot of new things like a label system. Every track can have several labels. I plan to add special label statistics which will show the total distance over all tracks for a label, average speed/distance/time per track for a label etc.

If you are interested in my future plans for the project, here are some points I consider implementing:

  • Track import/export
  • Waypoint import/export (especially .gpx files from GeoCaching
  • Supperpose the map over a scanned map image or automatic retreival of maps from the internet (e.g. Microsoft mapserver)
  • Add special features for sports training (logbook etc)
If you know something else which I should consider implementing, drop me a line or send a feature requests.

Just in case if you are wondering when I will do my first release: it will be soon. The pygarmin library has some problems right now and does not always work. We found the bug but could not fix it yet. Onces this is done I will pack pyTrack and there will be a Windows as well as Mac OS X version. A linux version is also possible, but I plan not to make packages. Still, I will provide some documentations on how to install it.

Welcome...

28 September 2004

This projects intends to create a cross platform utility to use garmin GPSs. It uses several different standard libraries to do so like wxPython, pySerial and pyGarmin.

Please feel free to submit feature requests so we know what you want the software to do.
